### Pre-flight Sequence

```
1. config.validate()               — required fields non-empty
2. run_validation(&dataset, ...)   — reuse JIN-169; bail on any error_count > 0
3. ModelRegistry::find(&model)     — skip HF check if already local
   └─ check_model_on_hub()         — HEAD https://huggingface.co/{model}
      └─ 404 → bail; 401 → OK (gated model still exists)
4. check_output_dir()              — create dir; check ≥10 GB free
5. [--dry-run] → print ETA stub, return early
6. write_train_script()            — write to NamedTempFile
7. spawn python3 <script> <args>   — stdout piped, stderr null or inherit
8. pipe stdout JSON line-by-line   — error → bail; done → print; step → verbose log
9. register_trained_model()        — upsert ModelEntry into models.json
```

### Subprocess Stdout Protocol

| JSON field | Meaning |
|---|---|
| `{step, epoch, loss, lr}` | Training step progress |
| `{event: "done", output: "..."}` | Training finished; output dir path |
| `{event: "interrupted", message}` | SIGINT caught; checkpoint saved |
| `{error: "...", message: "..."}` | Fatal error (e.g. no CUDA GPU) |

### TUI Architecture

```
commands/train.rs
  └─ preflight_and_spawn()          async pre-flight + spawn
  └─ run_train_tui(child, config, script)
       │
       ├─ build tokio::runtime (new_current_thread)
       ├─ tokio::spawn async reader task
       │    └─ AsyncBufReadExt on ChildStdout
       │    └─ parse JSON → send TrainEvent over std::sync::mpsc
       │
       └─ sync TUI event loop
            ├─ rt.block_on(yield_now())   — tick the reader task
            ├─ rx.try_recv()              — drain TrainEvent queue
            ├─ terminal.draw()            — render panel
            └─ crossterm::event::poll(250ms) — keyboard
```

### `total_steps` Inference

`total_steps` is not known at spawn time (it depends on dataset size). It is estimated from the first step that reports a non-zero epoch value:

```
steps_per_epoch ≈ step / epoch
total_steps = steps_per_epoch × total_epochs
```

This estimate is refined each tick and is accurate within one epoch's rounding.

### ETA Calculation

```
elapsed = start_time.elapsed().as_secs()
secs_per_step = elapsed / step
remaining = (total_steps - step) × secs_per_step
```

Formatted as `"Xs"`, `"Xm"`, or `"Xh Ym"` depending on magnitude.
