## Architecture Notes

| Item | Detail |
|---|---|
| Background approach | Moved from `body::before` animated pseudo-element to static `background-image` on `body` with `background-attachment: fixed`. Eliminates the `will-change` layer entirely — no GPU promotion cost at rest. |
| Depth hierarchy | `body` (ambient) → sidebar (float shadow) → main content wrapper (float shadow) → stat cards (card shadow) → topbar (1px glow). Every layer higher in Z has a tighter shadow. |
| Topbar glass | Topbar now has `backdrop-filter: blur(8px)`. This is a third backdrop-filter in the app (sidebar = blur 16px, chat panel = blur 16px, topbar = blur 8px). Topbar and sidebar are never visually overlapping, so composite layer cost is acceptable. |
| Light catch rule | Every elevated surface uses `inset 0 1px 0 oklch(100% 0 0 / 6–10%)` — simulates ambient light catching the top edge of a raised surface. Applied to: sidebar, stat cards, main content wrapper. |
| Orange glow rule | Orange elements (Cold Start card shadow, New Chat button) always emit `0 4px 12–24px oklch(0.7204 0.1507 48 / 12–30%)`. Consistent glow radius scales with element size. |
