## Keputusan Stack Final
- **Tauri 2** + React 18 + Vite 5 + TypeScript strict

## Design System Final

### Colors
```
bg-window:   #0f0d1a  ← darkest, window background
bg-sidebar:  #151222  ← sidebar solid
bg-content:  #181530  ← main content area
bg-card:     #1e1a2e  ← elevated cards
bg-hover:    #232040  ← hover state
orange:      #FF8C42  ← accent, disciplined use only
text:        #e8e6f0
text-muted:  #5a5670
```

### Spacing (Linear Scale)
```
2px / 4px / 6px / 8px / 12px / 16px / 20px / 24px / 32px
```

### Typography (Linear Scale)
```
11px — label, meta, uppercase
13px — body, nav items
14px — subheading
16px — heading, topbar title
20px — title
28px — stat values (Geist Mono)
```

### Radius
```
4px  — badge/pill
6px  — button
8px  — input
10px — card
12px — modal
16px — sidebar
```

---

## Sidebar Spec Final
- **Style:** Linear-inspired, floating, solid dark
- **Position:** `fixed`, gap `12px` semua sisi
- **Width:** 56px collapsed → 224px expanded
- **Toggle:** klik logo/chevron atas
- **NO glass, NO blur, NO border**
- **Shadow only:** `0 8px 40px oklch(0 0 0 / 50%)`

### Groups
```
[G Logo + chevron]

GENERAL
  Dashboard  (active = orange bg pill)
  Chat

AI TOOLKIT
  Models
  Train
  Dataset
  Eval

SYSTEM
  Doctor  (green/yellow/red health dot)

[spacer]

Settings  ← pinned bottom
```

---

## Card Rules
- **NO border** pada cards
- **Depth dari background color difference only**
- **NO shadow** (Linear style)
- **Border HANYA pada inputs** — `1px solid oklch(1 0 0 / 12%)`
- Input focus: `border-color: #FF8C42` at 60% + subtle glow ring
