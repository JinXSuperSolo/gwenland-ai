### Token Security Model

HF token is **never written to `config.json`**. Resolution order:

1. OS keyring (`gwen-cli` / `hf_token` entry via `keyring` crate)
2. `HF_TOKEN` environment variable
3. hf-hub default cache file (`~/.cache/huggingface/token`)

`gwen hub model login` stores in OS keyring only. `gwen hub model logout` removes it.

### `hub_pull()` internals

Uses `hf_hub::api::tokio::ApiBuilder::with_progress(show_progress)` to delegate progress bars to hf-hub's internal indicatif 0.18 (avoids version mismatch). Downloads every file listed in `repo_info.siblings` sequentially.

### `hub_push()` internals

Uses raw `reqwest` POST to `HF_ENDPOINT/api/models/<id>/upload/main/<filename>` with `X-Commit-Message` header and `application/octet-stream` body. `walkdir` collects files recursively; relative paths are normalised to forward-slashes for the URL.

### `hub_prune()` internals

Resolves `~/.cache/huggingface/hub/models--<owner>--<name>/` via `Repo::model().folder_name()`. Measures directory size with `dir_size()` before `remove_dir_all`.

### Server reachability check

```
probe_server(port)
  → true   — proceed to TUI / JSON mode
  → false  → server_unreachable_message(port)
              reads last_used_model from config.json
              prints: "⚠ No server running on port 1136. Start with:"
                      "  gwen serve --model mistralai/Mistral-7B-v0.1"
              exit code 3
```

### `stream_chat()` — SSE parsing flow

```
reqwest bytes_stream → byte_buf
  for each '\n'-terminated line:
    strip optional "data: " prefix
    if "[DONE]"    → session.finalize_assistant_turn() → ChatEvent::Done → return
    if valid JSON  → extract message.content → ChatEvent::Token
    if chunk.done  → session.finalize_assistant_turn() → ChatEvent::Done → return
    else           → treat as plain-text token
stream ends        → flush assistant_buf → ChatEvent::Done
```

### `--json` output format (AI agent consumption)

```jsonl
{"type":"token","content":"Sure"}
{"type":"token","content":"! Here"}
{"type":"token","content":"'s how..."}
{"type":"done","total_tokens":312}
```

On error:
```json
{"type":"error","message":"server returned HTTP 503"}
```
