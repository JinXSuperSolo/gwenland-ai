## Summary

Prepared the GwenLand monorepo for its first public GitHub push: canonical root layout,
MIT + Commons Clause licensing, layered `.gitignore` files, and relocation of session

### `gwen-cli/LICENSE.txt`

MIT + Commons Clause license for the GwenLand software package. Permits free use,
modification, distribution, and embedding. Restricts selling GwenLand itself (or
offering it substantially unchanged as a commercial hosted service) without permission.

`.env*`, agent folders, local scratch paths, Tauri generated state, postinstall binaries.
