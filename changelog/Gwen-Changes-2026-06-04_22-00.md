## Architecture Notes

| Item | Detail |
|---|---|
| SSE via fetch POST | Native `EventSource` only supports GET. `fetch` + `ReadableStream` + line-buffer handles SSE over POST correctly. Same pattern used for the Ollama pull NDJSON stream. |
| AbortController | Replaces the `reader.cancel()` approach. `abort()` tears down the TCP connection; `cancel()` only stops the JS read loop. `AbortError` is caught and silenced — partial message stays visible. |
| Dual SSE format | `useChat` handles both `{ done, token }` (GwenLand proxy) and `{ done, response }` (Ollama raw passthrough) so the hook works regardless of proxy version. |
| First-token latency | Measured as `performance.now()` delta from fetch start to first non-empty token. Stored in `ChatState.firstTokenLatencyMs`, displayed in SseBadge. |
| Proxy health poll | 3s interval, 2s timeout. 404 = alive (proxy running, `/health` not yet implemented). ECONNREFUSED / timeout = offline. Drives offline banner + send button disabled state. |
| `isActive` derivation | `OllamaModel.isActive` is not stored in raw state — it is derived per render as `model.name === activeModel`. Avoids a second `setState` call on every active-model change. |
| `refetch` pattern | Both `usePullModel` and `useDeleteModel` accept an `onSuccess` callback (bound to `useModels.refetch`) instead of importing `useModels` directly. Keeps hooks independently composable. |
| Drawer containment | `ModelInfoDrawer` uses `position: absolute` within a `position: relative` page container so it overlays the model list only, not the sidebar or window chrome. |
| Navigation option | `navigateTo(page, { openPull })` in App.tsx carries a one-shot option from Chat's ModelPicker to Models. Avoids lifting `showPull` state to App level. |
| Context length map | 15-family lookup table in `modelUtils.inferContextLength`. Prefix-match fallback covers variants like `llama3.1`, `llama3.2`. Unknown families default to 4096. |
| Syntax highlighting | Token-map regex in `CodeBlock` covers 5 categories (comment, string, keyword, function call, number) for Rust/JS/TS/bash. No external library. Zero bundle cost. |
