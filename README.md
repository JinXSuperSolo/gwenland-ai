## Quick start

```bash
cd gwen-cli
cargo build --release -p gwenland-tui
./target/release/gwenland --version
```

For the full guide — install, commands, GUI, training, and benchmarks — see **[gwen-cli/README.md](gwen-cli/README.md)**.
